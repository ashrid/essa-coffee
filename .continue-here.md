## Decisions Made

1. **Hybrid approach (Option 3+4)**: Email notifications on status changes + QR code for pickup
2. **QR token expiry**: 7 days with ability to reissue
3. **Auto-redirect**: After marking complete, redirect to scan page for next order
4. **Scan page content**: Name and order details only (no photo)
5. **Token generation**: Use `crypto.randomBytes(32)` for security
