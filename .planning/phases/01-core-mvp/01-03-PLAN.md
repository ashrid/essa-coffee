  <name>Task 1: Zustand cart store with localStorage persistence</name>
  <files>
    lib/cart-store.ts

  <action>
    Create `lib/cart-store.ts` using Zustand with `persist` middleware backed by localStorage:

    ```typescript
    import { create } from "zustand";
    import { persist } from "zustand/middleware";

    export interface CartItem {
      id: string;
      productId: string;
      name: string;
      price: number;
      image: string | null;
      slug: string;
      stockQuantity: number;
      quantity: number;
    }

    interface CartStore {
      items: CartItem[];
      isDrawerOpen: boolean;
      addItem: (product: Omit<CartItem, "quantity">) => void;
      removeItem: (productId: string) => void;
      updateQuantity: (productId: string, quantity: number) => void;
      clearCart: () => void;
      openDrawer: () => void;
      closeDrawer: () => void;
      totalItems: number;
      subtotal: number;
    }
    ```

    Implement:
    - `addItem`: if product already in cart, increment quantity by 1 (cap at stockQuantity). If new, add with quantity: 1.
    - `removeItem`: filter out item by productId.
    - `updateQuantity`: update quantity for productId. If quantity <= 0, remove item. Cap at stockQuantity.
    - `clearCart`: reset items to [].
    - `totalItems`: derived — `items.reduce((sum, item) => sum + item.quantity, 0)`. Implement as computed via `get` in the store (recalculate on read, not stored separately).
    - `subtotal`: `items.reduce((sum, item) => sum + item.price * item.quantity, 0)`.

    Use `persist` with name `"shop-seeds-cart"` and `storage: createJSONStorage(() => localStorage)`. Guard with `typeof window !== "undefined"` check for SSR safety.

    Export `useCartStore` as named export.

    Note: `totalItems` and `subtotal` should be computed selectors. With Zustand, implement them as getter properties in the state object or use `useCartStore.getState()` calls. The simplest approach: include `totalItems` and `subtotal` as regular state values that get updated in each action that modifies `items`.

    Also install sonner for toasts (already in package.json from Plan 01). In `app/(store)/layout.tsx`, add `<Toaster />` from sonner.
  </action>
  <verify>
    Open browser DevTools → Application → Local Storage → check `shop-seeds-cart` key appears after adding an item.
    Refresh page — cart data persists.
    Run TypeScript check: `npx tsc --noEmit` should pass.
  </verify>
  <done>useCartStore exported from lib/cart-store.ts. Cart persists to localStorage under key "shop-seeds-cart". addItem, removeItem, updateQuantity, clearCart, openDrawer, closeDrawer all implemented.</done>
