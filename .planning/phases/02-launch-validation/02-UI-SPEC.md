## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #fdf8f0 (cream-50) | Background, page surfaces |
| Secondary (30%) | #e8f0e8 (sage-100) | Cards, sidebar, nav backgrounds |
| Accent (10%) | #c8d8c8 (sage-200) | Focus states, highlights |
| Destructive | #dc2626 (red-600) | Destructive actions only |
| Primary | #1b4332 (forest-600) | Links, buttons, interactive elements |

**Accent reserved for:**
- Button focus rings
- Form input focus states
- Hover highlights on secondary UI
- Badge backgrounds (non-destructive)

**Source:** Phase 1.1 locked `tailwind.config.ts` with forest/sage/cream palette and `app/globals.css` with CSS variables. Phase 2 uses existing color tokens without modification.
